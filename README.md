## 🚀 How to deploy to GitHub Pages

1. **Create a new repository** on GitHub named exactly `my-portfolio`
2. **Upload `index.html`** to the root of the repository
3. **Create an `images/` folder** and upload your profile photo as `profile.jpg`
4. Go to **Settings → Pages**
5. Under **Source**, select `main` branch → `/ (root)` → click **Save**
6. Your site will be live at: `https://[your-username].github.io/my-portfolio`

---

## 📁 Folder structure

```
my-portfolio/
├── index.html        ← Main portfolio page
├── images/
│   └── profile.jpg   ← Your profile photo (add this!)
└── README.md
```

## 🖼️ Adding your profile photo

- Place your photo inside the `images/` folder
- Name it one of: `profile.jpg`, `profile.png`, `profile.jpeg`, `photo.jpg`, or `photo.png`
- The site will automatically detect and display it
- If no photo is found, your initials "AC" are shown as a stylish placeholder

---

## ✏️ Customisation

Open `index.html` in any text editor to update:
- **Social links** — add your LinkedIn/GitHub URL in the contact section
- **Projects section** — add any projects you've worked on
- **Colors** — edit the CSS variables at the top (`:root` block)
